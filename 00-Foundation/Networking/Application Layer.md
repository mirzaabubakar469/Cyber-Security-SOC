## 🔹 Important Application Layer Protocols

* HTTP 80 TCP – Web communication
* HTTPS 443 TCP– Secure web communication
* FTP – 20/21 TCP File transfer
* SMTP-25 TCP  – Sending emails
* POP3 110 TCP – Receiving emails
* DNS 53 UDP – Converts domain to IP
* SSH 23 TCP – Secure remote login
* Telnet 23 TCP – Remote login (not secure)
* SSL HTTPS Encryption
